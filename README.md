# 2026初级会计考试复习系统

## 强制登录同步模式

本系统已升级为强制登录同步模式,支持多设备实时数据同步。

---

## 快速开始

### 1. 部署后端
```bash
cd backend
wrangler deploy
```

### 2. 配置API地址
修改 `2026初级会计考试复习系统-完整版.html` 中的 API_BASE_URL:
```javascript
const API_BASE_URL = 'https://your-worker.workers.dev';
```

### 3. 开始使用
直接在浏览器中打开主HTML文件即可。

---

## 核心功能

- 🔐 **强制登录验证** - 未登录自动跳转
- 👤 **个人中心** - 用户信息、退出登录、切换账号、修改密码
- 🔄 **实时数据同步** - 学习进度和错题实时同步
- 📡 **网络状态监测** - 离线提示、自动刷新
- 🔑 **Token自动刷新** - 无需手动管理
